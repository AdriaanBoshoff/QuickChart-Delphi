# QuickChart Delphi
QuickChart client library for Delphi

Based on: https://github.com/typpo/quickchart-csharp

I would also like to clarify that the QuickChart.pas unit was mostly generated by Claude but modified to get it into a working state.

## Code Example

```delphi
  var chart := TChart.Create;
  try
    chart.Width := Trunc(img1.Width);
    chart.Height := Trunc(img1.Height);
    chart.Config := '''
            {
                type: 'line',
                data: {
                    labels: ['Q1', 'Q2', 'Q3', 'Q4'],
                    datasets: [{
                      label: 'Users',
                      data: [50, 60, 70, 180]
                    }]
                }
            }
            ''';

    chart.ToFile('.\test.png');

    img1.Bitmap.LoadFromFile('.\test.png');
  finally
    chart.Free;
  end;
```
